# agent-rules

A bit of a curious setup - I am linking `AGENTS.md` to `~/AGENTS.md`, but also to the `.codex/`, `.pi/`, `.claude/`, and `.copilot/` directories. This is because I want to have a single source of truth for the agent rules, but also want to have them available in the various agent directories for easy access.

I also link the whole `agent-rules/` directory to `~/agent-rules/` for easy access to the rules and any other related files.

PS: in the `AGENTS.md` I'm linking from the home folder, because whenever a thread is started, if I didn't specify $HOME, then it would search locally and fail to find the `@file`.
